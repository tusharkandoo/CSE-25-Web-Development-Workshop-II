## 🧪 Experiments

### Experiment 1: BMW Website ✅

**Objective:** Create a responsive automotive website using Bootstrap 5

</div>

**Features:**
- 🎨 Custom styled navigation bar with BMW logo
- 📱 Fully responsive design with mobile menu
- 🖼️ Hero section with carbon fiber background
- 🚗 Multiple BMW Series sections (3, 5, 7 Series)
- 🎯 Google Fonts integration (Black Ops One, Audiowide)
- 💫 Smooth hover effects and transitions

<div align="center">

**Technologies:** HTML5, CSS3, Bootstrap 5.3.8
