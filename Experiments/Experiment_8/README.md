# Experiment 8 - State Management using `useState`

This practical builds a simple React Counter App to understand state and event handling.

## Features

- Counter starts from `0`
- `+ Increment` button
- `- Decrement` button
- `Reset` button
- Step input (extra beginner-friendly enhancement)

## Concepts Used

- `useState` hook
- Event handling with `onClick` and `onChange`
- Dynamic UI update after state changes (re-render)

## Run

```bash
npm install
npm run dev
```

## Build Check

```bash
npm run build
```
