# Experiment 9 - State and Side Effects with `useEffect`

This practical demonstrates a simple registration form with validation and API data fetching.

## Features

- Controlled form inputs for name, email, and password
- Validation messages for empty or invalid input
- Success message after a valid registration
- API fetch using `useEffect`
- Display fetched users and registered users in separate cards

## Run

```bash
npm install
npm run dev
```

## Build Check

```bash
npm run build
```

## Notes

- The API users are fetched from `jsonplaceholder.typicode.com`
- The theme is light and print-friendly for lab submission
